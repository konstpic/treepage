# User guide

TreePage is a platform for reading and editing documentation synchronized from Git.

## Who this guide is for

| Role | What you can do |
|------|-----------------|
| **Viewer** | Read documents, search, download books |
| **Editor** | All of the above + edit, create pages, sync, create books |
| **Admin** (space) | Manage space members and repositories |

## Quick start for users

1. Open your organization's TreePage URL
2. [Sign in](../getting-started/first-login.md) via OIDC or use public access
3. Go to **Spaces** → select the one you need
4. Select a document in the tree on the left
5. Use **Search** to find information quickly

## Public spaces

If a space is marked as **public**, documents are available without login. On the home page, click **Open spaces**.

## Interface language

The interface language (English / Russian) is set globally by the administrator. When auto-translation is enabled, documents may display in the interface language.
