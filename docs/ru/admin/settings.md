# Системные настройки

**URL:** `/admin/settings`

> Запись настроек — только `super_admin`. Пользователи с ролью `admin` видят настройки в режиме только чтения.

## Язык интерфейса

Глобальный язык UI для всех пользователей.

| Язык | ID |
|------|-----|
| English | `en` |
| Русский | `ru` |

Применяется сразу при выборе (auto-save).

## Внешний вид (тема)

| Тема | ID | Описание |
|------|-----|----------|
| Fox White | `fox_white` | Светлая тема по умолчанию |
| Coral Night | `coral_night` | Тёмная с акцентами |
| Light | `light` | Классическая светлая |
| Dark | `dark` | Классическая тёмная |

## Автоперевод документов

| Параметр | Описание |
|----------|----------|
| Автоперевод документов на язык интерфейса (LLM) | Переводит документы и книги на язык UI |

**Требования:** LLM должен быть настроен (`LLM_ENABLED=true` на backend-server).

При выключении все пользователи видят оригинальный текст.

## Аутентификация

| Параметр | Production | Dev |
|----------|-----------|-----|
| Включить OIDC | ✅ | ✅ |
| Разрешить локальный вход | ❌ | ✅ |

Локальный вход работает только при `DEV_MODE=true` на backend-auth.

## Git-интеграция

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| Глобальная ссылка на токен | `GIT_ACCESS_TOKEN` | Env var для Git token |
| Webhook secret ref | `GIT_WEBHOOK_SECRET` | Env var для webhook secret |
| Интервал синхронизации (сек) | `300` | Default для новых репозиториев |
| Режим синхронизации | `scheduled` | Default sync mode |

## Платформа

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| Лимит поиска по умолчанию | `20` | Результатов на страницу |
| Макс. лимит поиска | `100` | Верхняя граница |
| Включить кэш | `false` | Redis/in-memory cache |
| Уровень логирования | `info` | debug / info / warn / error |

## Сохранение

Нажмите **Сохранить настройки** для секций Аутентификация, Git, Платформа.

Язык, тема и автоперевод сохраняются автоматически при изменении.

## Helm vs UI

Некоторые настройки дублируются в Helm values и UI:

| Настройка | Helm | UI |
|-----------|------|-----|
| OIDC issuer/client | `auth.oidc.*` | OIDC Providers |
| JWT TTL | `auth.jwt.*` | — |
| Search limits | `server.search.*` | Platform settings |
| Audit log | `server.security.enableAuditLog` | — |
| LLM | `server.extraEnv` | — |

Helm-настройки применяются при деплое. UI-настройки — в runtime через PostgreSQL (`system_settings`).
