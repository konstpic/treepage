# Быстрый старт

После [установки](../installation/README.md) TreePage выполните базовую настройку за 10–15 минут.

## Порядок действий

```
1. Первый вход          →  admin@local / admin (локальный super_admin)
2. Welcome space        →  /spaces/welcome — документация из docs/
3. Системные настройки  →  язык, тема, OIDC
4. Свои пространства    →  при необходимости
5. Git + sync           →  для вашей документации
6. Пользователи и RBAC  →  группы, роли
```

## Документы раздела

| Шаг | Документ |
|-----|----------|
| 1 | [Первый вход](first-login.md) |
| 2 | [Welcome space](welcome-space.md) |
| 3–6 | [Первоначальная настройка](initial-setup.md) |

## Роли при настройке

| Задача | Минимальная роль |
|--------|-----------------|
| Первый вход (dev) | — |
| Системные настройки, OIDC | `super_admin` |
| Создание пространств и репозиториев | `admin` |
| Синхронизация Git | `admin` или `editor` |
| Чтение документации | `viewer` или публичный доступ |

## Чеклист готовности к production

- [ ] Сменены секреты (`JWT_SECRET`, `DB_PASSWORD`)
- [ ] Настроен OIDC-провайдер
- [ ] Отключён dev-режим (`DEV_MODE=false`, `ENV=prod`)
- [ ] Welcome space доступен (`/spaces/welcome`)
- [ ] Выполнена первая синхронизация
- [ ] Назначены роли пользователям
- [ ] Настроен TLS на Ingress
