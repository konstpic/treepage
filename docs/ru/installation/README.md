# Установка TreePage

TreePage можно развернуть тремя способами — выберите подходящий для вашей среды.

| Способ | Когда использовать |
|--------|-------------------|
| [Docker Compose](docker-compose.md) | Быстрый старт, локальная разработка, демо |
| [Локальная разработка](local-development.md) | Разработка без Docker |
| [Kubernetes / Helm](kubernetes.md) | Production, GitOps, масштабирование |

## Общие требования

Перед установкой убедитесь, что выполнены [системные требования](requirements.md).

## После установки

1. Откройте веб-интерфейс (по умолчанию `http://localhost:5173` в dev).
2. Выполните [первый вход](../getting-started/first-login.md).
3. Пройдите [первоначальную настройку](../getting-started/initial-setup.md).

## Секреты

Независимо от способа установки задайте надёжные значения:

| Переменная | Назначение |
|------------|------------|
| `DB_PASSWORD` | Пароль PostgreSQL |
| `JWT_SECRET` | Подпись JWT-токенов |
| `OIDC_CLIENT_SECRET` | Секрет OIDC-клиента (production) |
| `GIT_ACCESS_TOKEN` | Токен доступа к Git-репозиториям |
| `GIT_WEBHOOK_SECRET` | Секрет для валидации webhook |

Подробнее: [Секреты и переменные окружения](../operator/secrets.md).
