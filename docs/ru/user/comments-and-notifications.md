# Комментарии и уведомления

> Требуется вход в систему. Комментарии доступны на странице каждого документа для авторизованных пользователей.

## Колонка комментариев

На странице документа (`/spaces/{slug}/docs/{doc-slug}`) справа отображается колонка **Комментарии** (на узких экранах — под текстом документа).

| Действие | Как |
|----------|-----|
| Читать обсуждение | Прокрутите панель комментариев |
| Оставить комментарий | Введите текст внизу → **Отправить** |
| Ответить коллеге | Используйте `@упоминание` в тексте |

Комментарии хранятся в TreePage (PostgreSQL), не в Git. На синхронизацию репозитория не влияют.

## @упоминания

Чтобы уведомить коллегу:

1. В поле комментария введите **`@`**
2. Появится выпадающий список активных пользователей (фильтруется по мере ввода)
3. Выберите пользователя или нажмите Enter — в текст подставится **email** (`@user@example.com`)
4. Отправьте комментарий

### Что получает упомянутый пользователь

- **Уведомление** в колокольчике (🔔) в шапке
- **Прямую ссылку** на документ и конкретный комментарий
- **Доступ на чтение** к этому документу, даже если он не участник закрытого пространства (упоминание даёт разовый доступ к странице)

Поддерживаются корпоративные email (`user@company.com`) и локальные учётки (`admin@local`).

## Уведомления

**Колокольчик** в верхней навигации показывает последние события.

| Тип | Описание |
|-----|----------|
| `comment.mention` | Вас упомянули в комментарии |
| События для редакторов | Новые документы, публикации, workflow (редакторы пространства) |

### Действия

| Действие | Описание |
|----------|-------------|
| Открыть уведомление | Клик по строке — для упоминаний открывается документ с прокруткой к комментарию (краткая подсветка) |
| Прочитано | ✓ у непрочитанного |
| Прочитать все | Ссылка в заголовке панели |

Избранное и полная история просмотров: **Мои страницы** (`/me`).

## Прямые ссылки на комментарии

Можно делиться якорем комментария:

```
/spaces/welcome/docs/getting-started#comment-{uuid}
```

При открытии URL с таким hash интерфейс прокручивает к комментарию и подсвечивает его.
