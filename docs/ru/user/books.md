# Книги (AI-сборки)

**Книги** — структурированные сборки документации из дерева страниц пространства. Текст глав берётся из реальных документов (без галлюцинаций). При подключённом LLM AI улучшает структуру и пишет введение.

> Создание и сборка книг требует роли **editor** или выше.

## Где найти

- В пространстве: вкладка **Книги** в боковой панели
- URL: `/spaces/{slug}/books`

## Типы аудитории

| Тип | Описание |
|-----|----------|
| Справочник разработчика | Developer reference с AI-структурой |
| Архитектурный обзор | Flowchart + Mermaid-диаграммы |
| Рунбук эксплуатации | Ops runbook |
| Онбординг (кратко) | Краткое введение для новых сотрудников |

## Создание книги

1. Откройте `/spaces/{slug}/books`
2. Заполните форму **Создать книгу из домена**:
   - **Фокус** (необязательно) — тема или поддерево документов
   - **Аудитория** — тип книги
3. Нажмите **Создать книгу**
4. Нажмите **Собрать книгу** для генерации глав

## Процесс сборки

```
1. TreePage выбирает релевантные документы из пространства
2. Формирует структуру глав
3. (Если LLM подключён) AI улучшает структуру и пишет введение
4. Текст глав — дословно из документации
5. Книга сохраняется в БД со статусом "готово"
```

### Статусы

| Статус | Описание |
|--------|----------|
| черновик | Создана, не собрана |
| сборка | Идёт генерация |
| готово | Можно читать и скачивать |
| ошибка | Сборка не удалась |

## Чтение книги

URL: `/spaces/{slug}/books/{book-slug}`

- Навигация по главам
- Скачивание в Markdown
- Индикатор «источники изменились» — если документы обновились после сборки

## Пересборка

- **Пересобрать** — обновить главы из актуальных документов
- **Заново** — полная перегенерация с LLM

## LLM

| Состояние | Поведение |
|-----------|-----------|
| LLM подключён | AI-структура + введение для developer/architect книг |
| LLM не настроен | Сборка работает, введение из описания книги |

### Настройка LLM (администратор)

Переменные окружения для `backend-server`:

```bash
LLM_ENABLED=true
LLM_API_URL=https://api.openai.com/v1   # или Ollama, vLLM и др.
LLM_API_KEY=sk-...
LLM_MODEL=gpt-4o-mini
```

## Автоперевод книг

При включённом **Автопереводе документов** (Настройки системы) книги также переводятся на язык интерфейса.
