# Поиск

TreePage поддерживает полнотекстовый поиск по документации.

## Как искать

1. Нажмите **Поиск** в навигации или перейдите на `/search`
2. Введите запрос
3. Нажмите **Искать** или Enter

### Прямая ссылка с запросом

```
/search?q=kubernetes&space=engineering&tags=helm,deployment
```

## Поля поиска

| Поле | Описание |
|------|----------|
| Заголовок | H1 и title документа |
| Содержимое | Текст Markdown |
| Теги | Frontmatter `tags:` |
| Репозиторий | Имя Git-репозитория |
| Автор | Автор последнего изменения |

## Фильтры

| Фильтр | Описание |
|--------|----------|
| Пространство | Ограничить поиск одним пространством |
| Автор | Фильтр по email автора |
| Теги | Через запятую (например `kubernetes, helm`) |

## Результаты

Каждый результат содержит:

- Заголовок документа (ссылка)
- Пространство
- Фрагмент текста с подсветкой совпадений
- Slug документа

## Ограничения

| Параметр | Значение по умолчанию |
|----------|----------------------|
| Лимит результатов | 20 |
| Максимальный лимит | 100 |

Настраивается администратором в **Настройки системы** → **Платформа**.

## Доступ

- Поиск по **публичным** пространствам доступен без входа
- Поиск по **приватным** пространствам — только после авторизации и с учётом RBAC

## Советы

- Используйте теги в Markdown для лучшей находимости:

  ```markdown
  tags: api, rest, authentication

  # API Reference
  ```

- Короткие запросы (1–2 слова) дают больше результатов
- Комбинируйте фильтр по пространству с текстовым запросом для точности
