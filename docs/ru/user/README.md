# Руководство пользователя

TreePage — платформа для чтения и редактирования документации, синхронизированной из Git.

## Для кого это руководство

| Роль | Что вы можете делать |
|------|---------------------|
| **Viewer** | Читать документы, искать, скачивать книги |
| **Editor** | Всё выше + редактировать, создавать страницы, синхронизировать, создавать книги |
| **Admin** (пространство) | Управлять участниками и репозиториями пространства |

## Быстрый старт для пользователя

1. Откройте URL TreePage вашей организации
2. [Войдите](../getting-started/first-login.md) через OIDC или используйте публичный доступ
3. Перейдите в **Пространства** → выберите нужное
4. Выберите документ в дереве слева
5. Используйте **Поиск** для быстрого нахождения информации

## Публичные пространства

Если пространство помечено как **публичное**, документы доступны без входа. На главной странице нажмите **Открыть пространства**.

## Язык интерфейса

Язык интерфейса (русский / английский) задаётся администратором глобально. При включённом автопереводе документы могут отображаться на языке интерфейса.
