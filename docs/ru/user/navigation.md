# Навигация по интерфейсу

## Главная страница (`/`)

Landing page с описанием возможностей TreePage. Кнопка **Открыть пространства** ведёт к списку пространств.

## Верхняя навигация

| Пункт | URL | Описание |
|-------|-----|----------|
| Пространства | `/spaces` | Список доступных пространств |
| Поиск | `/search` | Полнотекстовый поиск |
| Настройки | `/admin/*` | Панель администратора (только admin/super_admin) |
| Войти / Аккаунт | `/auth` | Вход и выход |

## Страница пространства (`/spaces/:slug`)

```
┌─────────────────────────────────────────────────────┐
│  ← К пространствам    Название пространства         │
├──────────────┬──────────────────────────────────────┤
│  Страницы    │  Содержимое документа                │
│  Книги       │  (Markdown + Mermaid)                │
│              │                                      │
│  📄 doc-1    │  Breadcrumbs: Space > Folder > Page  │
│  📁 folder/  │                                      │
│    📄 doc-2  │                                      │
│              │                                      │
│  [Sync]      │                                      │
└──────────────┴──────────────────────────────────────┘
```

### Боковая панель

- **Страницы** — дерево документов (вкладка по умолчанию)
- **Книги** — сохранённые AI-сборки
- **Синхронизировать** — ручной запуск sync (editors+)

### Режимы отображения пространств

На странице `/spaces` доступны три режима:

| Режим | Описание |
|-------|----------|
| Плитка | Карточки пространств |
| Таблица | Табличный вид с slug и доступом |
| Список | Компактный список |

## Страница документа (`/spaces/:slug/docs/:docSlug`)

- **Breadcrumbs** — навигационная цепочка
- **Редактировать** — переключение в режим редактора (editors+)
- **История версий** — просмотр предыдущих версий и diff

## Поиск (`/search`)

Строка запроса + фильтры по пространству, автору, тегам.

## Панель администратора (`/admin`)

Доступна пользователям с ролями `admin` или `super_admin`:

| Раздел | URL |
|--------|-----|
| Пространства | `/admin/spaces` |
| Репозитории | `/admin/repositories` |
| Пользователи | `/admin/users` |
| Группы | `/admin/groups` |
| Настройки системы | `/admin/settings` |
| OIDC-провайдеры | `/admin/oidc` |

## Темы оформления

Администратор задаёт глобальную тему. Доступные варианты: Fox White, Coral Night, Light, Dark.

## Горячие ссылки

- Документ: `/spaces/{slug}/docs/{doc-slug}`
- Книга: `/spaces/{slug}/books/{book-slug}`
- Поиск с запросом: `/search?q=kubernetes`
