# Обучение по интерфейсу

После **первого успешного входа** TreePage показывает краткий тур по интерфейсу. Сначала проигрывается приветственная анимация (логотип и приветствие) — тур запускается только после её завершения.

## Что показывает тур

| Шаг | Тема |
|-----|------|
| Навигация | **Пространства**, **Поиск**, **Аккаунт**, **Админка** (если вы администратор) |
| Разделы | Каталог пространств, форма поиска, **Мои страницы**, боковое меню админки |
| Документ | Дерево страниц и колонка **Комментарии** (на примере страницы) |

Кнопки **Далее** / **Назад** переключают шаги. **Пропустить** или **Готово** завершают обучение.

## Повторный запуск

В любой момент нажмите иконку **шапочки выпускника** (🎓) в верхней панели навигации.

## Техническая заметка

Факт прохождения сохраняется в браузере (`localStorage`, ключ `treepage_onboarding_v1_done`). Очистка данных сайта или другой браузер — тур снова предложится после входа.
